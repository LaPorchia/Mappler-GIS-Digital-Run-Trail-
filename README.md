Interactive Geospatial Mapping with Mappler
Overview

Last week in Dr. Eugene Levin’s CP Visualization and Unstructured Data course, we had the opportunity to learn from Dr. Wansoo Im, who introduced us to Mappler, an interactive geospatial visualization platform.

During the session, we explored how tabular datasets can be transformed into dynamic digital maps. By converting Excel data into geospatial visualizations, we were able to analyze and interact with location-based information in real time.

This repository contains sample datasets and visualizations created during that exercise. https://mappler.com/

Project Example

1. Nashville Running Trails

An interactive map displaying running trails across Nashville, using latitude and longitude coordinates.

This example demonstrates how geospatial tools can be used for:

Fitness and recreation mapping

Urban trail visualization

Location-based activity tracking

Why Mappler?

One of the most powerful aspects of Mappler is its direct connection between datasets and visualizations. When the underlying dataset is updated, the map automatically reflects those changes.

Key advantages include:

Real-time dataset updates

Interactive exploration of geographic data

User-controlled visualization layers

Accessible and dynamic data storytelling

Tools Used

Mappler – Geospatial visualization platform

Microsoft Excel – Dataset preparation

Latitude & Longitude Coordinates – Geospatial mapping
